# Projeto final PW3 | backend e painel administrativo da cantina

Inclui:
- API consumida pelo cardápio e pela cozinha (produtos, checkout, pedidos, status)
- Painel administrativo (produtos, estoque, pedidos, usuários, dashboards)
- Autenticação e controle de acesso (super admin / usuário)

**Backend central: expõe a API usada pelo cardápio (cantina-pedidos) e pela cozinha (cantina-cozinha).**
